# Telegram бот для скачивания медиа

Скачивает видео и музыку с YouTube, TikTok, Instagram и Яндекс.Музыки.

## Возможности

- Скачивание по ссылке (YouTube, TikTok, Instagram, Яндекс.Музыка)
- **Поиск с выбором** — показывает 5 результатов, вы выбираете нужный
- Работает в личных сообщениях и группах
- Очередь загрузок

## Файлы

```
bot.js       - код бота
package.json - зависимости
Dockerfile   - для Docker-деплоя
```

## Переменные окружения

| Переменная | Обязательная | Описание |
|------------|--------------|----------|
| BOT_TOKEN | Да | Токен бота от @BotFather |
| BOT_USERNAME | Нет | Имя бота (по умолчанию: lashmedia_pro_bot) |

## Деплой с Docker

```bash
docker build -t media-bot .
docker run -d -e BOT_TOKEN=ваш_токен media-bot
```

## Деплой без Docker

Требуется: Node.js 18+, yt-dlp, ffmpeg

```bash
npm install
BOT_TOKEN=ваш_токен node bot.js
```

## Команды бота

- `/start` - Начало работы
- `/help` - Помощь
- `/search <название>` - Поиск с выбором из списка
- `/status` - Статус бота

## Использование

1. **Ссылка** — отправьте ссылку с YouTube, TikTok, Instagram или Яндекс.Музыки
2. **Поиск** — напишите название песни, выберите из списка нужный трек
3. **В группах** — `!название` или `@botusername название`
